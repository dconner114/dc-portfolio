Survive the Storm

Shelter is a building game where you use the resources from each level to shelter a fire. Once the storm starts, you'll know if it was strong enough to protect the fire.

The Stack

Shelter uses the Godot game engine. All code is written in GDScript (Godot's native python-like scripting language). All assets were created in Figma. Sound effects were sourced from freesound.org.
