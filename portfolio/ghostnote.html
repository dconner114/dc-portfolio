A Secondhand Music Gear Marketplace

Ghostnote is a secondhand music gear marketplace built with Django for Harvard CS50w. It allows users to buy and sell used music gear.

Watch the video overview here:

The Stack

The backend is written in Python using Django to interface with a SQLite database. The frontend is built with HTML, CSS, and JavaScript. It uses Bootstrap for styling.

Features

Marketplace

Ghostnote operates much like eBay, where users can browse listings and purchase items.

Add your own listing

Users can create a listing for any item they want to sell.

Categories & Watchlists

Users can browse listings by category, or add items to their watchlist.
