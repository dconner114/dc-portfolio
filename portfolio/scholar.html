Desktop Study Tracking & Productivity

Scholar is a study tracking and productivity app built with Python and Tkinter. It allows users to track their study time, set goals, and create study sessions.

Watch the video overview here:

The Stack

Scholar is built using Node.js with a SQLite database. The frontend is comprised purely of html, css, and javascript. I chose to omit a frontend styling library as a challenge. I run the app in a Docker container so I can access it from my desktop when needed.

Features

Dashboard

The dashboard displays the user's study time, study sessions, courses, and a pomodoro timer.

New Entries

A user can add, delete, or edit study session and courses.
